# 📱 获取二维码的三种方法

## ✅ Expo 服务器正在运行
检测到 Expo 服务器正在端口 8081 上运行！

---

## 方法 1: 访问 Expo 开发工具网页（最简单）

### 步骤：

1. **打开浏览器**（Chrome、Edge 等）

2. **访问以下地址之一**：
   ```
   http://localhost:19002
   ```
   或
   ```
   http://localhost:19000
   ```

3. **你会看到一个网页**，上面有：
   - ✅ 二维码（可以扫描）
   - ✅ 连接 URL
   - ✅ 其他选项

4. **使用手机扫描二维码**：
   - iPhone：直接用相机应用扫描
   - Android：用 Expo Go 应用扫描

---

## 方法 2: 查看终端窗口

### 步骤：

1. **找到运行 Expo 的终端/命令行窗口**
   - 应该显示类似这样的内容：
   ```
   › Metro waiting on exp://...
   › Scan the QR code above with Expo Go (Android) or the Camera app (iOS)
   ```

2. **在终端输出中查找二维码**
   - 二维码是一个方形的 ASCII 图案
   - 通常在文本输出的中间部分

3. **如果看不到二维码**：
   - 尝试向下滚动终端窗口
   - 或者重启 Expo（见方法 3）

---

## 方法 3: 重新启动 Expo（查看完整输出）

如果你想在前台看到完整的输出和二维码：

1. **停止当前 Expo 服务器**：
   - 在终端窗口按 `Ctrl + C`

2. **重新启动（前台运行）**：
   ```powershell
   cd "D:\Habit Tracker\frontend"
   npm start
   ```

3. **现在你会看到完整的输出**，包括二维码

---

## 📲 扫描二维码的步骤

### iPhone/iPad：

1. **打开相机应用**
2. **对准二维码**
3. **点击屏幕上出现的通知**
4. **选择"在 Expo Go 中打开"**

### Android：

1. **打开 Expo Go 应用**
2. **点击"Scan QR code"或"扫描二维码"**
3. **允许相机权限**（如果提示）
4. **对准二维码扫描**

---

## 🔗 如果二维码不工作，使用 URL 连接

### 查找连接 URL：

在终端或网页界面中，找到类似这样的 URL：
```
exp://192.168.x.x:8081
```

### 手动输入：

1. **在 Expo Go 应用中**：
   - 点击 "Enter URL manually" 或 "手动输入 URL"
   - 输入完整的 exp:// URL

2. **对于 iPhone**：
   - 可以在 Safari 浏览器中输入 URL
   - 会提示在 Expo Go 中打开

---

## ⚠️ 重要提示

### 确保手机和电脑在同一网络：

- 📶 手机和电脑必须连接到**同一个 Wi-Fi 网络**
- 🔒 确保防火墙没有阻止连接
- 🌐 如果使用公司/学校网络，可能需要配置 VPN

### 如果仍然无法连接：

使用**隧道模式**（最可靠，但可能较慢）：
```powershell
cd "D:\Habit Tracker\frontend"
npx expo start --tunnel
```

隧道模式会创建一个可以通过互联网访问的 URL，不需要同一网络。
