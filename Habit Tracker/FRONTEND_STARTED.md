# 前端应用启动指南

## ✅ 前端应用已启动

Expo 开发服务器正在运行！

## 📱 如何访问应用

### 方式 1: 使用 Expo Go 应用（推荐）

1. **在手机上安装 Expo Go**
   - iOS: [App Store](https://apps.apple.com/app/expo-go/id982107779)
   - Android: [Google Play](https://play.google.com/store/apps/details?id=host.exp.exponent)

2. **扫描二维码**
   - 在终端中会显示一个二维码
   - 使用 Expo Go 应用扫描二维码
   - 应用将自动加载

### 方式 2: 使用模拟器

**Android 模拟器：**
- 在终端中按 `a` 键
- 或运行 `npm run android`（需要 Android Studio 和模拟器）

**iOS 模拟器（仅限 macOS）：**
- 在终端中按 `i` 键
- 或运行 `npm run ios`（需要 Xcode）

### 方式 3: 使用 Web 浏览器

- 在终端中按 `w` 键
- 或运行 `npm run web`
- 应用将在浏览器中打开

## 🔧 API 地址配置

**重要**: 如果使用真实设备测试，需要更新 API 地址：

### 当前配置
- 默认 API 地址：`http://localhost:3000`

### 使用真实设备时需要修改

1. 找到你的电脑 IP 地址：
   ```powershell
   ipconfig
   ```
   查找 "IPv4 地址"，例如：`192.168.1.100`

2. 修改 API 地址：

   编辑 `frontend/context/AuthContext.js`：
   ```javascript
   const API_BASE_URL = 'http://192.168.1.100:3000';  // 替换为你的 IP
   ```

   编辑 `frontend/services/api.js`：
   ```javascript
   const API_BASE_URL = 'http://192.168.1.100:3000';  // 替换为你的 IP
   ```

3. 重启 Expo 服务器

## 🎯 使用应用

### 首次使用

1. **注册账户**
   - 打开应用后，点击 "还没有账户？点击注册"
   - 填写：用户名、邮箱、密码（至少6个字符）
   - 点击 "注册"

2. **创建第一个习惯**
   - 登录后，点击主页右下角的 "+" 按钮
   - 输入习惯名称（例如："每天运动30分钟"）
   - 选择颜色和图标
   - 点击 "创建习惯"

3. **打卡**
   - 在主页点击习惯卡片上的 "打卡" 按钮
   - 或在习惯详情页点击 "今日打卡"

### 功能说明

- **主页**: 查看所有习惯，快速打卡
- **创建习惯**: 添加新的习惯追踪
- **习惯详情**: 查看统计信息、打卡记录
- **我的**: 查看个人资料和统计数据
- **设置**: 应用设置选项

## 🐛 常见问题

### 无法连接到后端

**症状**: 登录/注册时提示网络错误

**解决方案**:
1. 确保后端服务器正在运行（端口 3000）
2. 如果使用真实设备，检查 API 地址是否设置为你的电脑 IP
3. 确保设备和电脑在同一网络

### Expo Go 无法加载

**解决方案**:
1. 检查手机和电脑是否在同一网络
2. 尝试重启 Expo 服务器
3. 检查防火墙设置

### 二维码不显示

**解决方案**:
1. 在终端中按 `r` 重新加载
2. 检查网络连接
3. 尝试使用 URL 方式连接
