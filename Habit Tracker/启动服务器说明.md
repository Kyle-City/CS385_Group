# 启动服务器说明

## 前端和后端是不同的

- **后端（Backend）**: 运行在端口 3000，提供 API 服务
- **前端（Frontend）**: Expo 应用，连接到后端 API

## 启动后端服务器

### 在 backend 目录中运行：

```cmd
cd D:\Habit Tracker\backend
npm run dev
```

应该看到：
```
MongoDB connected
Server running on port 3000
```

## 启动前端应用

### 在 frontend 目录中运行：

```cmd
cd D:\Habit Tracker\frontend
npm start
```

或者清除缓存启动：
```cmd
npx expo start --clear
```

## 正确的启动顺序

1. **首先启动后端**：
   ```cmd
   cd D:\Habit Tracker\backend
   npm run dev
   ```
   （保持这个终端窗口运行）

2. **然后启动前端**（打开新的终端窗口）：
   ```cmd
   cd D:\Habit Tracker\frontend
   npm start
   ```

## 检查后端是否在运行

在 PowerShell 中运行：
```powershell
netstat -ano | Select-String ":3000" | Select-String "LISTENING"
```

如果看到输出，说明后端正在运行。

## 测试后端

在浏览器中访问：
```
http://localhost:3000/auth/me
```

应该返回错误（因为未认证），但能说明服务器在运行。
