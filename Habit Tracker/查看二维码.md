# 📱 查看 Expo 二维码的方法

## 🎯 推荐方法：使用浏览器（最简单）

### 步骤：

1. **打开浏览器**（Chrome、Edge、Firefox 等）

2. **访问以下地址**：
   ```
   http://localhost:19002
   ```

3. **你会看到 Expo 开发工具网页**，包括：
   - ✅ 清晰的二维码（可以扫描）
   - ✅ 连接 URL
   - ✅ 其他选项和按钮

4. **用手机扫描二维码**：
   - **iPhone**：直接用相机应用扫描二维码
   - **Android**：打开 Expo Go 应用，点击"扫描二维码"

---

## Terminal 方法：查看终端窗口

如果你想在终端中查看：

1. **找到运行 Expo 的终端/命令行窗口**
   - 应该有一个窗口显示了 `npm start` 的输出

2. **查看终端输出**
   - 二维码会显示为 ASCII 字符组成的方形图案
   - 通常在输出的中间部分
   - 可能需要在终端中向下滚动才能看到

3. **如果找不到终端窗口**：
   - 打开新的 PowerShell 或终端
   - 运行：
     ```powershell
     cd "D:\Habit Tracker\frontend"
     npm start
     ```
   - 现在会看到完整的输出，包括二维码

---

## 🔍 如何识别二维码

### 在终端中，二维码看起来像这样：
```
████████████████████████████████████
████████████████████████████████████
████ ▄▄▄▄▄ █▀█ █▄█▄▀█ ▄▄▄▄▄ ████
████ █   █ █▀▀▀█ ▄▀▀▀█ █   █ ████
████ █▄▄▄█ █▀ █▀▀█ ▀█ █▄▄▄█ ████
...
```

### 在浏览器中：
- 清晰的二维码图片，可以直接扫描

---

## 📲 扫描后如何使用

### iPhone：
1. 用相机扫描二维码
2. 点击屏幕顶部出现的通知
3. 选择"在 Expo Go 中打开"
4. 应用会自动加载

### Android：
1. 打开 Expo Go 应用
2. 点击"Scan QR code"（扫描二维码）
3. 允许相机权限
4. 扫描二维码
5. 应用会自动加载

---

## ⚠️ 如果无法连接

### 确保：
- ✅ 手机和电脑连接到**同一个 Wi-Fi 网络**
- ✅ Expo Go 应用已安装在手机上
- ✅ Expo 服务器正在运行

### 备用方案：手动输入 URL

如果在终端或网页中看到类似这样的 URL：
```
exp://192.168.1.100:8081
```

可以在 Expo Go 应用中手动输入这个 URL。

## 💡 快速检查

现在 Expo 服务器应该正在运行，你可以：

1. **立即尝试**：在浏览器打开 `http://localhost:19002`
2. **查看终端**：找到运行 `npm start` 的窗口
3. **扫描二维码**：使用手机扫描
